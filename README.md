> TSLint config for haifengfe


#### Getting Started

1. Install: `yarn add @haifeng-fe/eslint-config -D`.

2. Usage in your tslint.json:

    ```
    { "extends": "@haifeng-fe/tslint-config" }
    ```

    **OR**, if you are using the tslint.yml:
    ```yml
    extends: '@haifeng-fe/tslint-config'
    ```
